Natural By Design

Here's a succinct overview of the work I accomplished:




  1. ER Modeling: I was provided with a partially completed ER model and the task was to complete it. This involved the creation of new entities, attributes, and relationships, while also modifying existing ones as required. This task honed my data modeling skills and understanding of how entities relate within a system.
  2. SQL Database Implementation: The core of the project was to implement the designed database using SQL. I created a unique database named NBDDB-[MyInitials] and organized my SQL scripts in a sequence to ensure data integrity and database functionality. This involved testing if the database existed, creating the database, setting the database use, creating User-Defined Data Types (UDDTs), and creating tables.
  3. Database Constraints: A significant part of this project was enforcing data quality. I did this by applying appropriate CHECK constraints on each entity. This provided a level of data validation at the database level, enhancing the reliability of the data.
  4. Record Insertion: I populated the database with at least 5 records for each table, giving me hands-on experience with SQL data manipulation commands.
  5. Querying: Lastly, I demonstrated my ability to retrieve useful information from the database by writing SQL queries to reproduce two given Information Management System reports (IMS-2 and IMS-3).



This project underscored the critical role of database design and management in software development and sharpened my skills in ER modeling and SQL. The knowledge and skills I gained from this project have equipped me to handle complex database projects and turn raw data into useful, high-quality information.
